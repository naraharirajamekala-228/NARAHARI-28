<product_requirements>
**Product: MyApp - Group Car-Buying Platform**

**Core Idea:** A web application that enables users to form groups based on car brands to negotiate bulk discounts with dealers.

**User Flow & Features:**
1.  **Authentication:** Users can register and log in.
2.  **Group Discovery (Home Page):** Users see a list of available car brand groups (e.g., Tata, Mahindra). Each group card displays the brand logo, member count (e.g., 41/50), and a progress bar. Users can filter by brand and search.
3.  **Joining a Group:**
    *   On the Group Detail page, clicking Join opens a selection modal.
    *   The user must first select a specific car model, variant, and transmission type (Manual/Automatic).
    *   The system displays the on-road price for the selected configuration in Hyderabad.
    *   A dynamic joining fee is calculated based on price tiers (e.g., ₹1000 for cars <₹10L, ₹2000 for ₹10-20L, etc.).
    *   The user pays this dynamic fee via a mock Razorpay integration to join the group.
4.  **Group Details:** Once joined, a member can see a list of other members and their chosen car preferences (e.g., Jane Doe: Safari - Accomplished+ Automatic).
5.  **Admin & Negotiation:** An admin dashboard exists to view locked groups (when member count reaches max), add dealer offers, and view voting analytics.
</product_requirements>
